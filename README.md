# osmium

Web browser written from scratch in C++

<img src="https://i.ibb.co/MR0xyDm/serenity.png" width="700">

## Features
* Built from scratch
* Minimal (~700 LOC)
* Depends only on Qt
* Cross-platform
* DOM Inspector
* Proxy support

## Building
```bash
mkdir build && cd build
qmake ..
make
```
